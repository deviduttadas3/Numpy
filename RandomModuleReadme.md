<h1>Random Module In Python </h1>
`The Numpy Random Module is Used For:`
 <ul type="o">
       <li>generate random numbers</li>
        <li>create random arrays</li>
       <li> create datasets</li>
       <li> simulate AI data</li>
       </ul>

### Where is it Used:
<strong> Ai Models needed:</strong>
<ul type="o">
       <li>random data</li>
        <li>random weights</li>
       <li> random sampling</li>
       <li>random initialization</li>
       </ul>

### <b>Syntax:</b>
`np.random.randint(start, end)`
### EXAMPLE -->
<p>import numpy as np

num = np.random.randint(1,10)

print(num)
<p>Possible Output:
7</p> <br>
Every run may produce different output.
</p>
<h2>Important Note ⚠️</h2>
<p>Last value excluded.

Example:

np.random.randint(1,10)

generates:

1 → 9

NOT 10.</p>
